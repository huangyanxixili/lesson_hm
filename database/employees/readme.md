- 建表收集 姓名、工资、部门 等信息

- 选择合适的数据库
   - Mysql 关系型数据库 
       - 外建 建立表关联
       - 数据库的三大范式
           - 第一范式 列不可分
           - 第二范式 表中不能有重复数据
           - 第三范式 表中不能有冗余数据 
   - MongoDB 非关系型数据库 即NoSQL
    
   - 轻量级关系数据库 SQLite（安装npm i sqlite3）

- sqlite 
    SQLite 是一个轻量级的（Mysql 企业级）、嵌入式的（文件型）、使用SQL的数据库引擎，它不需要单独的服务器进程或系统配置。
